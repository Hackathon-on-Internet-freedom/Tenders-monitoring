# Tenders-monitoring
Есть наработки по мониторингу госзакупок, делал коммерческий проект, но можно перепрофилировать по нужды сообщества: поиск тендеров на площадках: госзакупки, сбер-аст, втб и телекомы (билайн, мтс, tele2) (проект изначально делался для мониторинга IT-тендеров). Так же есть движок, который может мониторить любые сайты на периодической основе, нужно только парсеры написать.
